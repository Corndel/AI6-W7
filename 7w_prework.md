## Create a WandB account and get your API key

WandB is a tool that tracks every model you train. By the end of the session you will have a dashboard showing all your runs side by side.

**Step 1.** Go to [wandb.ai](https://wandb.ai) and sign up. You can use your GitHub account to log in directly.

**Step 2.** Once signed in, go to [wandb.ai/settings](https://wandb.ai/settings).

**Step 3.** Scroll to the **API keys** section and click **New key**. Give it any name.

**Step 4.** Copy the key immediately and paste it somewhere safe : a notes app, a password manager, anywhere you can find it on the day. **You can only see the key once.** If you lose it, you can create a new one, but it is easier not to.

That key is a 40-character string starting with letters and numbers. It looks something like: `a1b2c3d4e5f6...`

---

## On the day

When you run the setup cell in the notebook, you will see a prompt:

```
wandb: Enter your API key:
```

Paste your key there and press Enter. That is it.
